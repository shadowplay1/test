** The object structure will look like this:

```js
{
    status: Boolean,
    value: { // object returns if reward is already claimed; else - number
        days: Number,
        hours: Number,
        minutes: Number,
        seconds: Number,
        milliseconds: Number
    },
    pretty: String | Number,
    reward: Number | Number[] // array returns if work reward is array
}
```
